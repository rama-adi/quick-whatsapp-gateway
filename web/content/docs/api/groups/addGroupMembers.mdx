---
title: Add members to a group
description: >-
  Adds the listed **participants** (user JIDs) to the group.


  **Capability:** requires `send`.


  **Precondition (live action):** the session must be **connected** and its
  account must be a group **admin**. If the session is not connected the gateway
  responds **501 `not_implemented`**.


  **Success:** **204 No Content** (empty body).


  **Notes:** WhatsApp may decline to add some numbers (privacy settings, invalid
  number) without failing the whole call. Adding a JID that is already a member
  is a no-op.


  **Errors:**

  - **400 `validation_error`** — empty participant list or a malformed JID.

  - **404 `not_found`** — the session or group does not exist (or is not yours).

  - **403 `forbidden`** — the session's account is not an admin of this group.

  - **501 `not_implemented`** — the session is not connected.
