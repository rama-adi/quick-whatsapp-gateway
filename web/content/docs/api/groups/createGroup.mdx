---
title: Create a group
description: >-
  Creates a new WhatsApp group with the given **name** (group subject) and
  starting **participants** (a list of user JIDs), and returns the new group's
  info.


  **Capability:** requires `send`.


  **Precondition (live action):** the session must be **connected**. Creating a
  group talks to WhatsApp in real time; if the session is not connected the
  gateway responds **501 `not_implemented`**.


  **Side effects:** the group is created on WhatsApp with this session's account
  as owner/first admin, the listed participants are invited, and the new group
  is upserted into the gateway's stored WhatsApp data so it appears in
  `listGroups` immediately. Not idempotent — calling twice creates two distinct
  groups.


  **Errors:**

  - **400 `validation_error`** — missing/empty name, no participants, or a
  malformed JID.

  - **404 `not_found`** — no session with this id is owned by your organization.

  - **501 `not_implemented`** — the session is not connected.
