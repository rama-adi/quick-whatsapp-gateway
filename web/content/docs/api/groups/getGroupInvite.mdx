          Returns the group's current invite link (a
          `https://chat.whatsapp.com/<code>` URL anyone can use to join), in the
          shape `{ "invite": "https://chat.whatsapp.com/..." }`.


          **Capability:** requires `read`.


          **Precondition (live lookup):** unlike most read endpoints this is a
          **live** WhatsApp query, so the session must be **connected**; if it
          is not the gateway responds **501 `not_implemented`**. The session's
          account must also be a group admin to fetch the link.


          **Errors:**

          - **404 `not_found`** — the session or group does not exist (or is not
          yours).

          - **403 `forbidden`** — the session's account is not an admin of this
          group.

          - **501 `not_implemented`** — the session is not connected.
