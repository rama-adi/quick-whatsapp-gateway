    contents:
      - content: >-
          Makes the member named by `jid` a group **admin**.


          **Capability:** requires `send`.


          **Precondition (live action):** the session must be **connected** and
          its account must already be a group **admin**. If the session is not
          connected the gateway responds **501 `not_implemented`**.


          **Success:** **204 No Content**. Promoting someone who is already an
          admin is a no-op.


          **Errors:**

          - **404 `not_found`** — the session, group, or member does not exist
          (or is not yours).

          - **403 `forbidden`** — the session's account is not an admin of this
          group.

          - **501 `not_implemented`** — the session is not connected.
