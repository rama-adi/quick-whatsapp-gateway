          List the webhook endpoints configured for the caller's organization.
          Requires the `manage` capability.


          The response is scoped to the caller's organization — webhooks owned
          by other organizations are never returned, and secrets are never
          included.


          This endpoint returns the full set in one page (the response carries
          an empty `nextCursor`); no pagination parameters are accepted. Errors:
          `403 forbidden` if the caller lacks the `manage` capability.
