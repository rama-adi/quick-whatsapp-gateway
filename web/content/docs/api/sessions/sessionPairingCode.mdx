          Get a phone pairing code so the session can be linked without QR
          scanning.


          Pass `phone` in E.164 digits (no plus sign). The returned code is
          short-lived and must be used from the phone.


          Requires `manage` capability; session must be unpaired.


          Errors: `validation_error` (400), `not_found` (404), `unauthorized`
          (401), `forbidden` (403).
