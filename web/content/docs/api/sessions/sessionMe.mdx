          Returns the WhatsApp identity of the number attached to this session —
          its JID, linked-device id, push name, and phone number.


          Requires the `manage` capability.


          **Precondition.** The session must be paired; if no number is attached
          yet, this returns `not_found` (404). On builds where this is not yet
          wired, it may return `not_implemented` (501).


          **Errors.** `not_found` (404) — session unknown or not yet paired.
          `not_implemented` (501) — the feature is stubbed in this build.
          `unauthorized` (401) / `forbidden` (403) — credential / capability
          failures.
