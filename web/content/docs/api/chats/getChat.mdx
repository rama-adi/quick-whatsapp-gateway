---
title: Get a chat
description: >-
  Returns one chat by its id (`cid`, the chat JID), served from the gateway's
  **stored copy**. Use this to read a single chat's archive/pin/mute flags and
  unread count.


  Requires the `read` capability.


  **Errors**


  - `not_found` (404): the session does not exist, is not owned by the caller's
  organization, or no chat with that `cid` is stored for the session.

  - `forbidden` (403): the caller lacks the `read` capability.
