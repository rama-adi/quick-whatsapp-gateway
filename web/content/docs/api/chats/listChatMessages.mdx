      - content: >-
          Returns a page of messages in the chat named by `cid`, served from the
          gateway's **stored copy** (the message history the gateway has
          ingested), newest first.


          Page through results with `limit` and `cursor`: send the `nextCursor`
          from the previous response back as `cursor` to fetch the next (older)
          page. When `nextCursor` is `null` there are no more messages.


          Requires the `read` capability.


          **Errors**


          - `not_found` (404): the session does not exist, is not owned by the
          caller's organization, or no chat with that `cid` is stored.

          - `forbidden` (403): the caller lacks the `read` capability.
