title: List chats
description: >-
  Returns a page of the session's chats, served from the gateway's **stored
  copy** (not a live WhatsApp query), ordered for display.


  Page through results with `limit` and `cursor`: send the `nextCursor` from the
  previous response back as `cursor` to fetch the next page. When `nextCursor`
  is `null` there are no more chats.


  Requires the `read` capability.


  **Errors**


  - `not_found` (404): the session does not exist or is not owned by the
  caller's organization.

  - `forbidden` (403): the caller lacks the `read` capability.
