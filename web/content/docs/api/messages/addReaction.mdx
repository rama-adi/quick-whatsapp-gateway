          Reacts to the message identified by `mid` (located by `chat`/`sender`
          in the body) with a single emoji.


          A reaction is **idempotent per message**: sending a new reaction
          replaces any reaction you previously set on that message — there is at
          most one reaction per (you, message) pair.


          Returns **200** with a `SendResult`.


          ### Preconditions & errors

          - Requires the **`send`** capability.

          - **400 `validation_error`** — missing/empty `emoji`.

          - **404 `not_found`** — the session or message does not exist or is
          not owned by your organization.
