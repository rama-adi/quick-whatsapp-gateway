---
title: Get a contact's about text
description: >-
  Fetches the contact's "about" text — the short status line shown on their
  profile — from WhatsApp.


  This is a **live lookup** against WhatsApp — the session must be
  **connected**. If the session is not connected the gateway responds `501`
  (`not_implemented`). The `about` field may be an empty string when the contact
  has none or has restricted it by privacy settings.


  **Auth:** requires the `read` capability.


  **Errors:** `404` (`not_found`) if the session does not exist or is not owned
  by the caller's organization; `501` (`not_implemented`) if the session is not
  connected.
