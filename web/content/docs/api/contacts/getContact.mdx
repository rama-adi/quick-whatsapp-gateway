      - content: >-
          Returns everything stored for one contact, addressed by its `lid`
          (WhatsApp's stable per-account identifier for a person): their name,
          whether you have a direct chat with them, and every group you have
          seen them in — including their nickname and role in each group.


          Served entirely from **stored** data, so no live WhatsApp connection
          is needed.


          **Auth:** requires the `read` capability.


          **Errors:** `404` (`not_found`) if the session does not exist, is not
          owned by the caller's organization, or no contact with the given `lid`
          is stored for that session.
