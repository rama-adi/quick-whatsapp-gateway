      - content: >-
          Start one in-memory backfill job for a single session.


          The job fetches contacts and joined group metadata/members. Chat
          history is not pulled by backfill.


          Returns **202 Accepted** with job state. Poll `GET
          /admin/sessions/{session}/backfill` for progress.


          Only one backfill can run per session; existing in-flight jobs return
          `conflict` (409).


          Requires platform super_admin login JWT. Api-keys and org roles are
          rejected with `forbidden` (403).


          Errors: `not_found` (404), `conflict` (409), `not_implemented` (501).
